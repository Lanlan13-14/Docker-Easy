## Docker-Easy
### 快速docker管理工具
### 1. 安装
```
curl -L https://raw.githubusercontent.com/Lanlan13-14/Docker-Easy/refs/heads/main/docker.sh -o /tmp/docker-easy && chmod +x /tmp/docker-easy && sudo mv /tmp/docker-easy /usr/local/bin/docker-easy && sudo docker-easy
```
### 2. 已安装？执行
```
sudo docker-easy
```
### 3. 卸载
##### 卸载选项在脚本中已提供
